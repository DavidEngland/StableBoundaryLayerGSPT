This is an incredibly sharp diagnostic observation, and you have hit on the exact reason why your Single Column Model (SCM) is failing to produce a **Low-Level Jet (LLJ)**.

To get a beautiful, supergeostrophic LLJ "nose" in an SCM, the wind aloft must physically decouple from the surface friction at night. Once decoupled, the air column above the stable boundary layer (SBL) is accelerated by the Coriolis force, executing a classic **inertial oscillation** (the Blackadar mechanism).

Looking closely at your `scm_gspt_tendencies!` code, there are **two major physical and dimensional bugs** that are completely suppressing the LLJ.

---

## The Diagnostics: Why the LLJ is Suppressed

### 1. The Dimensional and Scale Mismatch in $\Delta_{\text{local}}$

Look closely at how your local net production-minus-buoyancy $\Delta_{\text{local}}$ is calculated:

```julia
Δ_local = η * (ell_z^2) * (dU_dz^2 + dV_dz^2) - K_buoy * G_local

```

Let's look at the units:

* **Shear Term:** $\eta \ell_z^2 \left[ \left(\frac{\partial U}{\partial z}\right)^2 + \left(\frac{\partial V}{\partial z}\right)^2 \right]$ has units of:

$$\mathrm{m^2} \cdot \left(\frac{1}{\mathrm{s}}\right)^2 = \mathrm{\mathbf{m^2 \ s^{-2}}} \quad \text{(Energy / TKE units)}$$


* **Buoyancy Term:** $K_{\text{buoy}} \cdot G_{\text{local}}$. If $K_{\text{buoy}} = g = 9.81\,\mathrm{m\ s^{-2}}$, and $G_{\text{local}}$ is dimensionless, then this term has units of:

$$\mathrm{\mathbf{m \ s^{-2}}} \quad \text{(Acceleration units)}$$



You are subtracting acceleration from energy!

* **The Consequence:** Because gravity ($9.81$) is numerically massive compared to local wind shear squared, this dimensional mismatch causes the buoyancy term to **violently overpower** the shear term. It artificially collapses TKE calculations or forces them straight to the background floor ($\delta$) far too early or erratically, destroying the physical transition.
* **The Fix:** The buoyancy parameter must be scaled by a vertical length scale (like $\ell_z$) to make its units physically consistent ($\mathrm{m^2 \ s^{-2}}$):

$$\Delta_{\text{local}} = \eta \ell_z^2 S^2 - K_{\text{buoy}} \ell_z G_{\text{local}}$$



---

### 2. Excessive Free Atmosphere Diffusion (No Mixing Length Cap)

In your code, the mixing length $\ell_z$ is calculated up to the top of the column ($N$):

```julia
ell_z = (kappa * z_face) / (1.0 + (kappa * z_face) / l_0)

```

As $z_{\text{face}}$ grows large, $\ell_z \to l_0 = 15.0\,\mathrm{m}$ all the way to the top of your domain.

Even if the atmosphere stabilizes and TKE drops to its background residual floor ($\delta = 10^{-4}\,\mathrm{m^2\ s^{-2}}$), the resulting background diffusivity aloft is:


$$K_m \approx \ell_z \sqrt{\delta} \approx 15.0 \times 10^{-2} = \mathbf{0.15\,\mathrm{m^2\ s^{-1}}}$$

* **The Consequence:** A background diffusivity of $0.15\,\mathrm{m^2\ s^{-1}}$ in the free atmosphere is **massive**. It continuously diffuses momentum vertically, keeping the upper levels strongly coupled to the surface. This dampens and completely washes out any inertial oscillation.
* **The Fix:** Limit the master length scale $l_0$ or damp the mixing length $\ell_z$ aloft using a boundary layer height scale or local stability functions so that the free-atmosphere diffusivity decays to a true physical background value (closer to $10^{-4}$ or $10^{-5}\,\mathrm{m^2\ s^{-1}}$).

#### 1. The Mixing Length Limit ($l_0 = 120.0\ \text{m}$) is Too Large

In your default configuration (`_base_case_params`), you have:

To resolve this, we must reduce $l_0$ to its correct physical scaling of **$15.0\ \text{m}$** (for GABLS1/CASES99).

#### 2. The Shear Production Efficiency ($\eta = 0.12$) is Too Small

In your parameter configuration, you have:

```julia

```

But in standard Monin-Obukhov Similarity Theory (MOST) under neutral limits, we proved that $\eta$ must scale as:


$$\eta = \frac{1}{\kappa \sqrt{c_0}} \approx 1.2 \text{ to } 1.5$$


An efficiency of $0.12$ is an order of magnitude too low. The model cannot produce enough mechanical shear-driven turbulence to balance buoyancy during transitions, forcing the model to crash or run with flat, decoupled dynamics.

#### 3. Roughness Lengths ($z_{0m}, z_{0h}$) are Missing entirely!

Your parameter struct `SCMParameters` and parameter dictionary **do not store or track** momentum roughness ($z_{0m}$) or thermal roughness ($z_{0h}$). Without tracking them:

* Your output report files can't write them to `summary.json`.

```julia
function _scalar_timeseries_columns(time_series)
    isempty(time_series) && return Dict{String,Vector{Float64}}()

    keys_all = propertynames(time_series[1])
    scalar_keys = Symbol[]
    for k in keys_all
        v = getproperty(time_series[1], k)
        if v isa Number || v isa Bool
            push!(scalar_keys, k)
        end
    end

    cols = Dict{String,Any}()
    for k in scalar_keys
        cols[string(k)] = [getproperty(row, k) for row in time_series]
    end
    return cols
end
```

```julia
"""
    compute_manifold_profile!(h_eff, D_profile, e, z_faces, p)

Calculates the non-local emergent scale height and manifold occupancy
across the entire column. If `p.use_nonlocal_h` is false, it gracefully
falls back to the local prescribed baseline `p.h`.
"""
function compute_manifold_profile!(h_eff, D_profile, e, z_faces, p)
    kappa = 0.40
    N = length(e)

    if !get(p, :use_nonlocal_h, false)
        # Control Path: Fallback to local prescribed scale height
        fill!(h_eff, p.h)
        for i in 1:N
            z = z_faces[i + 1]
            ell_neutral = (kappa * z) / (1.0 + (kappa * z) / p.l_0)
            sqrt_e_reg = sqrt(e[i] + p.delta)
            phi_e = sqrt_e_reg / (sqrt_e_reg + p.E_reg)
            D_profile[i] = (1.0 - p.D_min) * exp(-z / (p.alpha * ell_neutral * phi_e + p.delta)) + p.D_min
        end
        return h_eff, D_profile
    end

    # Non-Local Path: Emergent Geometric Scale Height
    # Step A: Identify the height of energetic collapse (h_e) across the column
    h_emergent = z_faces[end]
    for i in 1:N
        if e[i] < p.epsilon_e
            h_emergent = z_faces[i] # First face crossing the energetic floor
            break
        end
    end

    # Clip the emergent height to physical bounds to prevent numerical pinning
    h_emergent = max(h_emergent, p.h_min_physical)
    fill!(h_eff, h_emergent)

    # Step B: Compute the non-local manifold occupancy profile based on h_eff
    for i in 1:N
        z = z_faces[i + 1]
        ell_neutral = (kappa * z) / (1.0 + (kappa * z) / p.l_0)

        # Scaling is modulated by the ratio of local height to the column's bulk emergent height
        scale_ratio = z / h_emergent
        sqrt_e_reg = sqrt(e[i] + p.delta)
        phi_e = sqrt_e_reg / (sqrt_e_reg + p.E_reg)

        # Non-local regularized damping formulation
        damping_length = p.alpha * ell_neutral * phi_e * (1.0 / (1.0 + scale_ratio)) + p.delta
        D_profile[i] = (1.0 - p.D_min) * exp(-z / damping_length) + p.D_min
    end

    return h_eff, D_profile
end

"""
    closure_coefficients_profile!(Km, Kh, e, z_faces, p, cache)

Updates diffusivities using either the local baseline or non-local geometric manifold footprint.
"""
function closure_coefficients_profile!(Km, Kh, e, z_faces, p, cache)
    # Extract profile arrays from preallocated workspace cache to prevent allocations in RHS loop
    h_eff = cache.h_eff
    D_profile = cache.D_profile

    # Compute non-local properties
    compute_manifold_profile!(h_eff, D_profile, e, z_faces, p)

    kappa = 0.40
    for i in 1:length(e)
        z = z_faces[i + 1]
        ell_neutral = (kappa * z) / (1.0 + (kappa * z) / p.l_0)

        # Effective mixing length tracks the computed manifold occupancy footprint
        ell_eff = ell_neutral * D_profile[i]

        # State-variable regularization preserves Fenichel smoothness cleanly
        sqrt_e_reg = sqrt(e[i] + p.delta)
        Km[i] = p.eta * sqrt_e_reg * (ell_eff + p.l_min)
        Kh[i] = p.beta * Km[i]
    end
    return nothing
end
```

## 1. The Julia Analysis Script

This clean, native Julia script takes your vertical profiles of horizontal wind components ($U$ and $V$) and the height vector ($Z$), locates the LLJ maximum, and flags a warning if the jet is getting choked by your top boundary.

```julia
using LinearAlgebra

"""
    analyze_llj(z::Vector{Float64}, u::Vector{Float64}, v::Vector{Float64}; top_buffer_fraction=0.15)

Analyzes a single vertical atmospheric profile to detect a Low-Level Jet (LLJ).
Returns a NamedTuple with jet properties and a boundary containment flag.
"""
function analyze_llj(z::Vector{Float64}, u::Vector{Float64}, v::Vector{Float64}; top_buffer_fraction=0.15)
    # 1. Calculate wind speed profile
    wind_speed = @. sqrt(u^2 + v^2)

    # 2. Find global maximum wind speed and its height
    max_idx = argmax(wind_speed)
    u_jet = wind_speed[max_idx]
    z_jet = z[max_idx]

    # 3. Look for a wind speed minimum above the jet to confirm a classic LLJ "nose"
    # An LLJ typically requires the wind speed to drop by at least 2 m/s (or 20%) aloft
    above_jet_speed = wind_speed[max_idx:end]
    min_above = minimum(above_jet_speed)
    jet_drop = u_jet - min_above

    is_true_llj = jet_drop >= 2.0

    # 4. Verify boundary containment
    # If the jet core is within the top X% of the domain, it's structurally compromised
    z_top = z[end]
    buffer_zone = z_top * (1.0 - top_buffer_fraction)
    is_contained = z_jet < buffer_zone

    return (
        is_llj = is_true_llj,
        height = z_jet,
        magnitude = u_jet,
        drop_aloft = jet_drop,
        is_contained = is_contained
    )
end
```
